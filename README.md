# java-cli-gradle-jacoco-jbehave-junit5-hello-world

## Description
A POC app using junit5,
jacoco, jbehave framework.

## Tech stack
- java
- gradle
  - junit5
  - jbehave
  - jacoco

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
